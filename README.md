# Autonomous-Embedded-Vehicle
Self-driving vehicle with embedded systems.
- Raspberry Pi
- Arduino
- Camera
- Ultrasonic Sensor
- Wheel Encoder



Relevant Skills:
- PD control
- Lane detection
- Obstacle detection
- Path planning
- Concurrent(multi-threaded) programming
